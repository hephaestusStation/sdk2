class jaxon_client.client.Client(base_url: Optional[str] = None, client_id: Optional[str] = None, client_secret: Optional[str] = None)[source]

Bases: object

The client contains information for connecting to a running Jaxon service It provides functions for creating projects, datasets and running various workflows on imported datasets such as labeling, training models. The models and the curated datasets can be exported via this client.

__init__(base_url: Optional[str] = None, client_id: Optional[str] = None, client_secret: Optional[str] = None)[source]

Initialize the JaxonClient to access the Jaxon server.

To successfully create a client a valid base_url must be provided.

>>> import jaxon_client as jc
>>> client = jc.Client(base_url="http://localhost:31415", client_id="user_name", client_secret="password")
Parameters

  • base_url (str) the base URL to a running Jaxon service.

  • client_id (str) the id to authenticate.

  • client_secret (str) the secret for the id to authenticate
